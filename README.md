# 🚫 Animated 404 Error Page

A simple and visually appealing **Animated 404 Error Page** built using **HTML, CSS, and JavaScript**.  
This project displays a custom "Page Not Found" screen with smooth animations to enhance user experience.

---

## 📌 Features

- ✨ Smooth animations
- 🎨 Clean and modern UI
- ⚡ Lightweight and fast
- 📱 Responsive design
- 🔧 Easy to customize

---

## 🛠️ Technologies Used

- HTML5
- CSS3 (Animations & Styling)
- JavaScript (Interaction)

---

Animated-404-error/<br>
